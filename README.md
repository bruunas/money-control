Money Control
=============

A dashboard to financial control.


## Available Scripts

In the project directory, you can run `yarn` or `npm`.

#### `yarn start`

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.


#### `yarn build`

Builds the app for production to the `build` folder.


## Technologies

* React
* React Hooks
* Styled Components
